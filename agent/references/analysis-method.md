# 六维快速分析框架

> 用途：盘中紧急快速分析 | 时间：3-5分钟 | 风格：超短线右侧趋势跟随

---

## 分析原则

1. **速度优先**：3-5分钟必须出结论，数据不全先标"暂缺"，绝不恋战
2. **右侧唯一**：只做确认后的趋势，不做预判、不抄底、不赌反转
3. **结论为王**：少写描述，直接给判断——买/观察/弃牌，一句话说清逻辑
4. **止损铁律**：每票必带止损位，趋势破坏无条件离场，不补仓、不幻想
5. **赔率导向**：向上空间必须大于向下空间（至少2:1），否则不做

---

## 六维框架

### 维度1：趋势强度（权重35%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 均线排列 | 5/10/20/60日多头排列+价格站于5日线上方 | 多头排列+8起评，死叉或破60日线直接3分以下 |
| 价格位置 | 处于近期新高附近或突破平台 | 突破近期高点+2，跌破10日线-3 |
| 高低点结构 | 低点抬高+高点抬高 | 符合上升趋势+1，结构破坏归零 |

- **关键判断**：趋势方向（↑→↓）+ 趋势强度评分（1-10）
- **弃牌红线**：价格跌破20日线且均线死叉，直接0分

### 维度2：资金动能（权重25%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 主力资金 | 当日主力净流入占比>5%或连续3日净流入 | 主力净流入+3，净流出-3 |
| 成交量 | 成交量较5日均量放大1.5倍以上 | 放量+2，缩量-2 |
| 换手率 | 换手率处于近期活跃区间（3%-20%） | 适中+2，过低或过高酌情扣分 |
| 北向资金 | 近3日北向净流入该板块/个股 | 流入+1，流出-1 |

- **关键判断**：资金流入还是流出，动能评分（1-10）
- **弃牌红线**：主力连续3日净流出且缩量，直接0分

### 维度3：动量信号（权重15%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 近期涨幅 | 近5日涨幅10%-30%为宜（已启动但未高潮） | 10%-20%+3，>30%追高风险-2，<5%无动能-3 |
| 涨停基因 | 近20日有过涨停或连续涨停记录 | 有涨停+2，无涨停-1 |
| 突破形态 | 放量突破前高/平台/整理区间 | 确认突破+3，假突破-3 |
| 技术指标 | RSI(6)在50-80区间，MACD红柱放大 | RSI超买>80-2，MACD金叉+2 |

- **关键判断**：动量是否充足，是否处于加速段
- **弃牌红线**：RSI>90严重超买且MACD顶背离，直接0分

### 维度4：概念热度（权重12%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 板块排名 | 所属概念板块当日涨幅排名前20 | 前10+3，前20+1，未入榜-2 |
| 涨停家数 | 同概念涨停家数≥3家（有板块效应） | ≥5家+3，<3家-1 |
| 龙头地位 | 是否为本概念领涨龙头或前排跟风 | 龙头+3，中排+1，后排-1 |
| 催化持续性 | 概念催化是否有持续发酵预期（>3天） | 有持续催化+1，一日游-2 |

- **关键判断**：概念是否处于市场焦点
- **弃牌红线**：板块无涨停且排名50开外，直接0分

### 维度5：消息催化（权重8%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 消息质量 | 有实质性利好（政策/订单/重组/业绩超预期） | 实质性利好+3，朦胧消息+1，无消息-1 |
| 兑现时间 | 3天内有可兑现的催化剂（发布会/政策落地/业绩披露） | 3天内+2，1周内+1，无近期催化-1 |
| 消息层级 | 国家级/行业级/公司级 | 国家级+2，行业级+1，公司级+0 |
| 预期差 | 消息是否已被市场充分定价 | 未充分定价+2，已涨停兑现-3 |

- **关键判断**：消息是驱动力还是噪音
- **弃牌红线**：利好已完全兑现（如已涨停开板放量），直接0分

### 维度6：基本面速扫（权重5%）

| 检查项 | 判定标准 | 评分（1-10） |
|--------|----------|-------------|
| 业绩雷排查 | 非ST/*ST，近季度无亏损，无退市风险警示 | 有ST风险直接0分 |
| 估值极端 | PE<200或PB<20（超短线可放宽，但排除极端） | 极端高估-2，正常范围+1 |
| 业绩窗口 | 是否处于业绩披露敏感期（1/4/7/10月） | 业绩期+1（可能有超预期），业绩雷区-2 |
| 流通盘 | 流通市值适中（20-200亿），易于资金拉动 | 适中+1，过大（>500亿）-1 |

- **关键判断**：是否有硬伤及是否处于业绩窗口期
- **弃牌红线**：ST/*ST/停牌/退市风险，直接0分

---

## 评分规则

- **总分** = 趋势×35% + 资金×25% + 动量×15% + 概念×12% + 消息×8% + 基本面×5%
- **四舍五入取整**，最终得分0-100

| 分数段 | 操作指令 |
|--------|----------|
| <60分 | **不看/不买/不分析，直接弃牌**——浪费时间是超短线最大的成本 |
| 60-70分 | **观察，不操作**——等信号更明确或等回调再评估 |
| 70-85分 | **符合条件，给出具体入场/止损/止盈建议**——标准仓位参与 |
| >85分 | **强烈推荐，仓位可适当加重（不超过总仓位30%）**——确定性较高的机会 |

## 输出模板

### 快速分析结论格式

```
【结论】买入/观察/弃牌 — 一句话说清逻辑
【评分】XX分（趋势X/资金X/动量X/概念X/消息X/基本面X）
【关键信号】1. ... 2. ... 3. ...
【止损位】XX元（-X%）
【赔率评估】向上X% vs 向下X%，赔率约X:1
【催化剂】近X天内可关注的触发事件
【风险提示】最大风险点
```

### 示例

```
【结论】买入 — 板块龙头放量突破前高，资金持续流入，动量充足
【评分】82分（趋势9/资金8/动量8/概念9/消息7/基本面6）
【关键信号】1. 放量涨停突破近期平台 2. 所属概念板块涨幅第3，同板块5家涨停 3. 主力连续3日净流入，换手率12%活跃
【止损位】15.80元（-6%）——跌破10日线坚决离场
【赔率评估】向上18% vs 向下6%，赔率约3:1
【催化剂】明日下午行业政策发布会，利好预期未完全兑现
【风险提示】若发布会不及预期，可能高开低走；RSI已75，短期超买需警惕
```

## 快速模式执行纪律

| 纪律项 | 执行标准 |
|--------|----------|
| 数据来源 | 以本地Parquet为主，`mx-skills`/`wencai-skills`仅补充关键缺失字段 |
| 分析深度 | **不画图，不做深度财务分析**，只看关键指标和信号 |
| 输出内容 | **只给结论+关键数据+止损+赔率**，不写长篇 reasoning |
| 超时处理 | 如果5分钟内数据无法获取，直接标注"数据暂缺"并降低置信度 |
| 止损执行 | 触发止损位**无条件离场**，不犹豫、不补仓、不等反弹 |
| 仓位纪律 | 单票不超过总仓位30%，评分>85分方可顶格 |
| 复盘要求 | 当日收盘后10分钟快速复盘：止损是否该触发？评分哪里失真？ |

## 附：3分钟极速版（应急用）

当用户说"快""紧急""盘中"时，跳过完整六维，只检查以下5项：

1. **趋势**：价格是否在5日线上方且均线多头排列？（否→弃牌）
2. **资金**：当日是否放量且主力净流入？（否→弃牌）
3. **动量**：近5日涨幅10%-30%且有过涨停？（否→弃牌）
4. **概念**：所属板块当日是否有≥3家涨停？（否→弃牌）
5. **硬伤**：是否ST/业绩雷/已完全兑现？（是→弃牌）

**5项全过→直接给评分+止损+赔率，30秒输出结论。**

---

> **核心心法：超短线的本质是概率游戏，框架的作用是快速识别高概率机会，然后机械执行。不追求每把都赢，追求赢的时候多赚、输的时候少亏。**

# 参考文献目录

本目录存放 Agent 运行时参考的技术文档。

## 文件说明

| 文件 | 说明 |
|------|------|
| `analysis-method.md` | 六维快速分析框架（轻量级分析） |
| `deep-dive-workflow.md` | 十三维度深度分析框架（V2 Fusion） |

## 内容来源

- `analysis-method.md`：六维快速框架，适用于超短线快速研判
- `deep-dive-workflow.md`：基于《上市公司深度拆解工作流》融合 V2 Fusion 框架

## 使用方式

Agent 在执行深度分析时自动引用 `deep-dive-workflow.md` 中的方法论；
执行快速分析时引用 `analysis-method.md`。
